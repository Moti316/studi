## עקרון-זכויות

| מקור-החומר                     | מעמד-זכויות                   | כלל-טיפול בקורס                                                                           |
| ------------------------------ | ----------------------------- | ----------------------------------------------------------------------------------------- |
| חוקים · תקנות · פקודות · צווים | **נחלת-הכלל** (חקיקה ממשלתית) | **שימוש-ישיר** — ציטוט-נוסח מותר; PDF = source-of-truth (ADR-005/011)                     |
| פסקי-דין · הנחיות-מינהל רשמיות | נחלת-הכלל (פרסום-רשמי)        | שימוש-ישיר כחומר-הוראה; לציין מקור רשמי                                                   |
| תקני-ISO · ת"י · IEC           | **בתשלום — לא נחלת-כלל**      | **תקצירים-מקוריים בלבד** · אסור לשכפל/לצטט טקסט-תקן (ראה 5.x ב-LEGISLATION-SOURCES)       |
| מצגות/סיכומי-מרצה              | זכות-יוצרים של היוצר          | **reference + שכתוב-מהותי** — לא לשכפל; לנסח מחדש בלשון-הפלטפורמה                         |
| בנקי-מבחן ממותגים (©)          | זכות-יוצרים מסומנת במפורש     | **בקשת-הרשאה / השמטה** — לא לשלב כפי-שהם (ראה § פתוח-להכרעה)                              |
| תוכן-משתמש (עבודת-גמר אמיתית)  | פרטיות-משתמש                  | גיבוי-reference פרטי; כל פלט-נגזר name-clean (ראה [`FINAL-PROJECT.md`](FINAL-PROJECT.md)) |

**עיקרון-על:** שמות-אדם (מרצה/תלמיד/מנחה/חתימה) **מוסרים מכל פלט** — גם כשהמקור נחלת-הכלל.
חומר נחלת-הכלל = חופשי לתוכן; שם-אדם = לעולם לא בפלט.

## פתוח להכרעת-מוטי

קבוצות שמצב-הזכויות שלהן עמום או ממותג — **החלטה מקרה-במקרה** (לא להטמיע עד הכרעה):

| קבוצה                               | הסוגיה                                                                            | אופציות-טיפול                                                   |
| ----------------------------------- | --------------------------------------------------------------------------------- | --------------------------------------------------------------- |
| בנקי-מבחן-לדוגמה ממותגים (4 מודלים) | מסומנים **© מפורש** של יוצר + מותג-מסחרי; ~400 שאלות עם-תשובות, כיסוי-רוחבי מצוין | 🟠 בקשת-הרשאה · או 🔴 השמטה + יצירת בנק-מקביל מקורי מהחקיקה     |
| עבודת-גמר-לדוגמה (פרויקט-אמיתי)     | מתעדת מפעל/אתר **אמיתי** + שמות-תלמיד/מנחה; דרושה כתבנית ל-capstone               | 🟠 גיבוי-reference פרטי בלבד + שכתוב name-clean כתבנית-נייטרלית |
| שו"ת/שאלות-ועדה גולמיות             | חוברו מזיכרון-נבחנים (ללא יוצר-מסומן); מעמד-זכויות לא-ברור                        | 🔵 סיכום-מקורי / שכתוב · או 🟣 reference-בלבד                   |
| מסמכי-מרצה image-only (OCR)         | תוכן לא-נקרא; אי-אפשר להעריך זכויות/כיסוי עד OCR                                  | 🟣 reference + OCR → הערכה-מחודשת לפני-שכתוב                    |
| מותג מסחרי על מצגת-חזרה             | מצגת-חזרה כללית נושאת מיתוג-חברה בכותרת                                           | 🟢 שכתוב-מהותי + הסרת-מיתוג · או 🟠 הרשאה אם משמרים-כפי-שהם     |
| חוברות-שדה רשמיות (פנקס/קטלוג)      | מונפקות ע"י גוף-בטיחות/תקינה; reuse עשוי לדרוש-ייחוס                              | 🟣 reference + ייחוס-לגוף-המנפיק (לא שם-אדם)                    |

> **דגש name-clean:** גם בקבוצות שמוכרעות ל"שכתוב-מהותי" — כל שם-אדם (מרצה/תלמיד/מנחה/חתימה)
> **מוסר**. הכלל חל על השכתוב והפלט, לא על גיבוי-reference פרטי של מקור (ADR-005/011).
