## שלוש שכבות-נכס פר-תקנה (ללא שכתוב)

1. **`.md` (כאן) = נוסח-עדכני נבו verbatim** — קורפוס-עבודה/RAG + מאתר-ציטוט. נמשך אוטומטית מנבו (HTML) ע"י [`scripts/fetch-legislation.ts`](../../../../scripts/fetch-legislation.ts) (מניפסט ב-[`scripts/legislation-manifest.ts`](../../../../scripts/legislation-manifest.ts)). חילוץ **דטרמיניסטי לא-גנרטיבי** (regex, לא WebFetch/AI → אפס סיכון-שכתוב).
2. **PDF ב-Drive = המקור-המחייב המלא** (כולל תוספות/טבלאות/איורים). תיקיית-Drive "חוקים ותקנות"; כל `.md` מצביע אליו ב-`authoritative_source`. **creator-gated** (הרשאה — למוטי/אימות, לא ללומד).
3. **תכנית-הלימודים (מינהל-הבטיחות 905018)** מגדירה את ה-scope; שדה `depth` (ליבה/מסגרת/ענפי) ב-frontmatter.

> **`INDEX.md` מחולל אוטומטית** (`pnpm legislation:index`) מהמניפסט — מקשר פר-scope: `.md ↔ PDF-Drive ↔ נבו ↔ עומק`. אפס-דריפט.

### ערובת-נאמנות ("מילה-במילה") של ה-`.md`
ה-extractor **לא-גנרטיבי** → לא משכתב, רק יכול להשמיט/להוסיף בגבולות. הבקרות:
- **L1** (פנימי): רצף-טוקנים זהה לחילוץ-פרמיסיבי (zero-loss) — אחרת שגיאה.
- **L2–L5** (`pnpm legislation:verify`): רציפות-מספור · גבולות+אין-chrome · שלמות-עברית (charset) · frontmatter.
- **L1-חי** (`--verify-live`): diff-טוקנים מול נבו — אומת **42/42 identical**.

### ⚠️ תוכן-תמונה (12 נוסחים, `source_complete: false`)
נבו מטמיע תוספות/טבלאות/איורים/נוסחאות כ-**תמונות base64** (לא טקסט) → ה-`.md` חסר אותן (גלאי-תמונות אוטומטי מסמן `source_complete:false` + `gap_note`). **הנוסח-המלא ב-PDF-המחייב ב-Drive.** מסומנים: 1.1 · 2.0 · 2.3 · 2.4.1 · 2.6 · 2.8.1 · 2.10b · 3.1 · 3.2 · 3.4 · 3.5.1 · 3.5.2.
