## דרישת משרד-העבודה

עבודת-הגמר היא **תנאי-הסמכה** במסלול הקורס-המוכר (לצד הבחינה-העיונית והבחינה בעל-פה מול הוועדה).
הדרישה המהותית — שהמשתמש יוכיח **יכולת-יישום** של ידע-הבטיחות על מקום-עבודה ממשי, ולא ידע-תאורטי
בלבד.

| דרישה             | פירוט                                                                    |
| ----------------- | ------------------------------------------------------------------------ |
| **אתר אמיתי**     | מפעל / אתר-בנייה / מקום-עבודה קיים — **לא תרחיש פיקטיבי**.               |
| **תצפית-שטח**     | סקר-מפגעים מבוסס תצפית ישירה במקום, לא תיאור-מהזיכרון.                   |
| **תכנית-בטיחות**  | תוצר-הליבה: תכנית-לניהול-הבטיחות לאתר (scope **1.2**, תקנות תשע"ג-2013). |
| **הערכת-סיכונים** | סקר-סיכונים שיטתי (JSA / מטריצת-סיכון) המבוסס על מפגעים שנצפו.           |
| **המלצות-בקרה**   | מדרג-בקרות (Hierarchy of Controls) — מהמסוכן-ביותר (חיסול) לפחות (PPE).  |
| **הגשה-מתועדת**   | מסמך-גמר מובנֶה + נספחים (תצלומים / שרטוטים / טפסים), כפי שתדרוש הוועדה. |

> **רפורמת-2021** (+12h בתכנית) נוספה בדיוק כדי שבוגר יֵדע **לכתוב** תכנית-לניהול-הבטיחות —
> כשירות-ליבה שהוועדה בוחנת בעבודת-הגמר. ראה דגש ניהול-הסיכונים ב-[`MOLSA-PROGRAM.md`](MOLSA-PROGRAM.md).

## נושאי-הפרויקט (8 אפשרויות רשמיות)

עבודת-הגמר מוגשת באחד מ-8 הנושאים (לפי הנחיות-המשרד; מאושרת ע"י מרכז-הקורס/מרצה-מאשר, וניתנת לבדיקת-מדגם ע"י המינהל):

1. פיתוח פתרון-טכני לבעיה בטיחותית.
2. תכנית לניהול תקציב-הבטיחות במפעל.
3. ניתוח-מעמיק של אירוע-בטיחותי + מסקנות ליישום ולקחים.
4. **ניהול-סיכונים** באחת ממחלקות-הייצור במפעל.
5. הכנת תכנית-חירום במפעל.
6. תכנית ארגון-בטיחותי של אתר-בנייה.
7. הכנת נהלים למפעל.
8. נושא-אחר בתחום הבטיחות-והבריאות-התעסוקתית, באישור מרכז-הנושא במטה.

> **המסלול המעוגן: ניהול-סיכונים (#4)** — שעבורו המשרד הגדיר **פורמט-JSA רשמי** (למטה), והוא הליבה של מבנה-הליווי. שאר הנושאים נתמכים אך פחות-מובְנים.

## פורמט-JSA רשמי + מבנה-מסמך (עדכון-תכנית 19.10.2025)

עדכון תכנית-הלימודים (פרסום 12.08.2025 · תוקף **19.10.2025**) הוסיף **פורמט אחיד לניתוח-סיכונים תעסוקתי (JSA)** — ובלשון-המשרד "הפורמט ישמש להכנת עבודת-הגמר". מבחן-ההסמכה כולל כעת גם שאלות-אמריקאיות + **תרגיל ניתוח-סיכונים**.

**עמודות-טבלת-ה-JSA (פורמט-החובה):** גורם-הסיכון · תרחיש-להתממשות · בקרות-קיימות (מנהלתיות / הנדסיות / **צמ"א**) · הערכת-סיכון (חומרה × סבירות → רמת-סיכון) · בקרות-נוספות-נדרשות · אחראי-לביצוע · תאריך-ביצוע · סטטוס.

**מטריצת-הסיכון (4×4):** רמת-סיכון = חומרה (1–4) × סבירות (1–4). מקרא-הכרעה: **1–4 ליבק** (טיפול-שגרתי) · **6–9 לא-ליבק** (להורידו; המשך-זמני רק אם הכרחי) · **12–16 לא-ליבק** (פעולה-מיידית, גם בהפסקת-עבודה). סיכון שהחוק אוסר — אינו "ליבק".

**מבנה-מסמך-ההגשה (6 חלקים):** 1. אודות-החברה (פרופיל) · 2. אודות-הפרויקט ותהליכי-העבודה · 3. מבנה-ארגוני · 4. פירוט תהליכי-עבודה ומפגעים · 5. טבלת הערכת-סיכונים ("לוח-החלטה" + מקרא) · 6. ניתוח-סיכונים.

**טמפלייטים רשמיים** (Drive: חומרי-לימוד/"פרויקט גמר" `1k1u…`): `המלצה לפורמט-טבלה בניתוח-הסיכונים` (פורמט-JSA) · `עמוד-פתיחה לפרויקט` · `לוח-החלטה לסקר-סיכונים` (מטריצת-הסיכון) · `פרויקטים.pdf` (הנחיות + עבודת-דוגמה — ⚠️ **PII, reference-פנימי בלבד**).

## מבנה-הליווי (שלבים)

המיני-קורס מלווה את המשתמש שלב-אחר-שלב; כל שלב הוא תוצר-ביניים שהמערכת בודקת ומשׁבחת לפני המעבר
הלאה. כל תוצר-משתמש נשמר **name-clean** (תיאור-תפקיד בלבד — "מנהל-האתר", "העובד", "המעסיק" —
לעולם ללא שם-אדם).

| #   | שלב                            | תיאור                                                                              | scope/פרק       | סטטוס |
| --- | ------------------------------ | ---------------------------------------------------------------------------------- | --------------- | ----- |
| 1   | **בחירת-אתר אמיתי**            | המשתמש בוחר מפעל/אתר ממשי; המערכת מסייעת לאַפיין ענף, גודל, סיכונים-עיקריים.       | פרק א/ג         | ✅    |
| 2   | **סקר-מפגעים**                 | סיור-שטח מובנֶה; רשימת-תיוג מפגעים לפי תחנות-עבודה (חשמל / גובה / חומ"ס / מכונות). | פרקים ד/ה/ו/ז   | ✅    |
| 3   | **הערכת-סיכונים (JSA/מטריצה)** | לכל מפגע: חומרה × הסתברות → דירוג-סיכון; ניתוח-משימה (JSA) למשימות-המסוכנות.       | 6.x · פרק ט     | ✅    |
| 4   | **תכנית-לניהול-הבטיחות**       | תוצר-הליבה: תכנית-לניהול-הבטיחות לאתר לפי תקנות תשע"ג-2013 (מבנה-החובה של התקנות). | **1.2** · פרק ג | ✅    |
| 5   | **המלצות-בקרה (מדרג-בקרות)**   | לכל סיכון-מהותי: בקרה לפי המדרג — חיסול → החלפה → הנדסי → מנהלי → PPE.             | 5.6 · 6.x       | ✅    |
| 6   | **תיעוד והגשה**                | הרכבת מסמך-הגמר + נספחים; בדיקת-שלמות מול דרישות-הוועדה לפני הגשה.                 | פרק י"א         | ✅    |

**מקרא:** ✅ שלב-מוגדר · 🟠 דורש-חיזוק · ⬜ ממתין-לתוכן.

### דגשים-מתודולוגיים

- **שלב 4 הוא הלב.** התכנית-לניהול-הבטיחות (scope 1.2) מכוסה היטב בחומרי-הלימוד, כולל תבנית-מילוי
  יישומית ("טמפלייט תכנית-בטיחות") — המערכת מַנחה את המשתמש למלא אותה על-בסיס תוצרי-שלבים 2–3 שלו.
- **רצף-תלות:** כל שלב נשען על קודמו (מפגעים → דירוג-סיכון → בקרות → תכנית). המערכת אינה מאפשרת
  לדלג על שלב-ביניים — זה מבטיח שעבודת-הגמר מבוססת-נתונים-אמיתיים ולא נכתבת "מהסוף".
- **מדרג-הבקרות** (שלב 5) הוא קריטריון-איכות מרכזי שהוועדה בוחנת: המלצה ל-PPE כאשר ניתן היה
  חיסול/בקרה-הנדסית — היא ליקוי. המערכת מתריעה על המלצות שאינן עולות-במדרג.

## חיבור ל-scope ולפרקים

מיני-קורס ה-capstone אינו פרק-תוכן נוסף, אלא **מצב-יצירה שמשלב** את הפרקים שכבר נלמדו לכדי תוצר
אחד. שלושת העוגנים:

| עוגן               | תפקיד בפרויקט-הגמר                                                              |
| ------------------ | ------------------------------------------------------------------------------- |
| **פרק י"א** (אתגר) | הנחיות עבודת-הגמר + המבחנים — ה-scaffolding הפורמלי של הליווי (ראה curriculum). |
| **scope 1.2**      | תכנית-לניהול-הבטיחות — תוצר-הליבה; מבנה-החובה לפי תקנות תשע"ג-2013.             |
| **scope 6.x**      | שיטות הערכת-הסיכונים (JSA / FMEA / Bow-Tie / מטריצה) — הכלים לשלבים 3 ו-5.      |

נגיעות-נוספות: פרקים ד–ח מספקים את **קטלוג-המפגעים** לפי-ענף (חשמל / חומ"ס / גובה-ובנייה /
חקלאות); פרק ט (ניהול-סיכונים) מספק את **מסגרת-העבודה** (45001 + מדרג-הבקרות). כך כל פרק-תוכן
"נפרע" כיישום-מעשי בעבודת-הגמר.
