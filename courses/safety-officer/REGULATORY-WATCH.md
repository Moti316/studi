## (א) תקנות ארגון הפיקוח על העבודה (ממונים על הבטיחות), תשפ"ה-2025 🟠

> תיקון מקיף לתקנות-הממונים הוותיקות (תשנ"ו-1996). נחתם ע"י שר-העבודה ונכנס לתוקף **24.6.2025**.
> משנה את **דרישות-ההסמכה עצמן** — לכן רלוונטי ל-meta-content (נושא-ועדה אפשרי "מהם תנאי-ההסמכה?").

| נושא                 | לפני (תשנ"ו-1996)           | אחרי (תשפ"ה-2025)                                                         |
| -------------------- | --------------------------- | ------------------------------------------------------------------------- |
| מסלולי-כשירות        | תואר+2ש' / הנדסאי-טכנאי+3ש' | **+ מסלול הנדסאי/מה"ט חדש** (דיפלומה מאושרת + התמחות/ניסיון + בחינת-ועדה) |
| ימי-השתלמות (כשירות) | 8 ימים **בכל שנה**          | **12 ימים על-פני 3 שנים** (ממוצע ~4/שנה — הקלת-עומס)                      |
| תוקף אישור-הכשירות   | (לפי הנוסח-הקודם)           | **3 שנים**, חידוש ל-3 שנים נוספות בכל פעם, מותנה השלמת-השתלמויות          |
| סנקציה אי-השתלמות    | —                           | היעדר-השתלמות מעל **8 שנים** → חובת בחינה/הכשרה מחדש                      |
| שקיפות               | —                           | **מרשם/מאגר ציבורי פתוח** של ממונים בעלי-כשירות בתוקף                     |
| דיווח                | שנתי-אישי                   | **דיווח תקופתי/מרוכז** (אפשר ע"י מוסדות-ההכשרה)                           |

- **מסלול הנדסאי/מה"ט (חדש):** דיפלומת-הנדסאי במסלול מאושר ע"י המכון-הממשלתי-להכשרה-בטכנולוגיה-ומדע
  (מה"ט) + **התמחות 18 חודשים** בתחום-הבטיחות תחת פיקוח ממונה-ותיק (3 שנות-ניסיון+), **או** 3 שנות
  ניסיון-מקצועי — בתוספת הצלחה ב**בחינה מקצועית מול ועדה** (לפחות פעמיים בשנה).
- **הוראות-מעבר:** ממונים-קיימים שומרים על תוקף-האישור עד 3 שנים מהדיווח-האחרון, בכפוף להשלמת 12
  ימי-השתלמות לפי הסטנדרט-החדש לפני החידוש.

## (ב) רפורמת הבטיחות בעבודות-בנייה, תשפ"ה-2025 🟠

> תיקון תקנות הבטיחות בעבודה (עבודות בנייה). פורסם ברשומות — **קובץ-התקנות 12061, 16.10.2025**
> (אושר בוועדה ~10.9.2025). **מועד-תחילה: כ-12 חודשים לאחר הפרסום — סביבות 10/2026.** מעבירה
> אחריות מהותית ליזמים/מזמיני-עבודה וקבלנים. נוגעת לפרק גובה+בנייה ולפרק סיכונים-ובקרות.

| רכיב                  | פירוט                                                                                                                             |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| בקר-בטיחות (חדש)      | מהנדס/הנדסאי-אזרחי/ממונה מוסמך מטעם **מזמין-העבודה**; ביקור באתר **אחת ל-3 חודשים**; דיווח על סיכונים בלתי-קבילים תוך **48 שעות** |
| מנהל-אתר (חדש)        | נדרש באתרים בשטח **15,000 מ"ר ומעלה** (לפי היתר-הבנייה), בנוסף ל"מנהל-עבודה" הממונה לפני תחילת-העבודות                            |
| תוכנית-בטיחות לכל-אתר | ללא-קשר-לגודל (בעבר רק מ-50 עובדים): עד 49 → תכנית-ארגון-בטיחות; 50+ → תכנית-ניהול-בטיחות; עדכון לפי התקדמות-הבנייה               |
| דיווחים מקוונים       | למפקח-העבודה-האזורי על התחלה/סיום/החלפת בעלי-תפקידים; תיעוד בפנקס-הכללי ובאתר                                                     |
| אחריות יזם/מזמין      | **אחריות פלילית ומנהלית/אזרחית** באי-מינוי-בקר, אי-הכנת-תוכנית או אי-טיפול-בסיכונים                                               |
