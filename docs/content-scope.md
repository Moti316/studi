## Implementation Notes

### Scope-filter ב-import pipeline

כל chunk שמיובא מ-Drive/megen חייב לכלול metadata:

```yaml
in_scope: true | false
scope_refs:
  - id: '1.1' # תקנות ארגון הפיקוח (ממונים)
    section: 'סעיף 3' # אופציונלי
  - id: '2.1' # עבודה בגובה
status: '[מאומת]' # | '[מוסקנא]' | '[לא ידוע]'
```

**כלל-מתחייב**:

- `in_scope=true` → eligible לשאלות-וועדה
- `in_scope=false` → נשמר כ-reference במנגנון אך לא עולה ל-quiz בפועל
- `[לא ידוע]` חסום בכל מקרה (כלל מ-ADR-005)

### Coverage tracker

מסך admin: `/admin/coverage` — מציג לכל אחד מ-21 הפריטים:

- מספר questions שיש (יעד: ≥ 5 per ID)
- מספר scenarios (יעד: ≥ 2 per ID)
- אחוז recall של מוטי

מבטיח שאף ID לא נופל בין הכיסאות לפני הוועדה.

### Out-of-Scope Content — תוכן שזוהה ב-Drive מחוץ ל-21

> ראה פירוט ב-`docs/content-inventory.md` § תוכן שמחוץ ל-scope.

הקבצים הבאים זוהו ב-Drive אך אינם חלק מ-21 פריטי-החקיקה. **לא יעלו ל-quiz**.

| תוכן                       | מקור ב-Drive       | גודל    | סטטוס                                                           |
| -------------------------- | ------------------ | ------- | --------------------------------------------------------------- |
| חוזק חומרים + תהליכי-ייצור | טל גרינהויז (docx) | 6 MB    | `in_scope=false`                                                |
| מערכות אוטומטיות + רובוטים | טל גרינהויז (docx) | 26 MB   | `in_scope=false`                                                |
| ריתוך וחיתוך               | מיכאל פייזל (PDF)  | 4.4 MB  | `in_scope=false`                                                |
| עיבוד שבבי                 | מיכאל פייזל (PDF)  | 2.3 MB  | `in_scope=false`                                                |
| דרגנוע + מעליות            | טל גרינהויז (docx) | ~43 MB  | `in_scope=false` (מוזכר ב-"חזרה להסמכה 2" עמ' 46-47 אך לא ב-21) |
| מצגת חזרה כללית — שאדי     | חומרי לימוד (PDF)  | 26 MB   | `in_scope=mixed` — chunking + filter נדרש                       |
| מצגת שיעור 1 — אייל פלטק   | אייל פלטק (PDF)    | 25.3 MB | `in_scope=mixed` — chunking + filter נדרש                       |

**כלל**: קבצים אלה נשמרים ב-`chunks` עם `status='out_of_scope'`. יוצגו כ-"הסבר לעומק" בלבד — אם רלוונטיים להסבר שאלה — אך לא יייצרו שאלות.
