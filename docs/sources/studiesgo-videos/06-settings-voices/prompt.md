# 🎬 סרטון 06 — Settings + Voice Preview

> **משך**: 37 שניות · **גודל**: 14MB

מסך-הקולות הוא ה-feature הקריטי-של-TTS.

## 📋 פרומפט לג'מיני

```
אני בונה אפליקציית-לימוד בעברית. אתה מקבל סרטון של מסך-הגדרות
ב-StudiesGo, עם סקציה מיוחדת לקולות TTS (יואב / טלי / מיכל / אורי).

אנא תן תיאור-בעברית-מפורט של:

1. **מבנה-מסך הגדרות** (סמן MM:SS):
   - איזה סקציות מופיעות (יעד-יומי / ערכה / קול / נגישות / חשבון)?
   - איך מסומנים section-headers?
   - האם יש icons לכל סקציה?

2. **Toggles + Switches**:
   - איך נראה toggle on vs off (צבע, אנימציה)?
   - האם יש sound/haptic feedback בלחיצה?

3. **בחירת-יעד-יומי**:
   - איך מוצגות אפשרויות (XP-קל / רגיל / אתגרי)?
   - מה האנימציה של הבחירה?

4. **בחירת-ערכה (Theme)**:
   - light vs dark mode toggle — איך נראה?
   - האם המעבר חלק? יש fade?

5. **מסך-קולות (הסקציה החשובה!)**:
   - 4 אופציות: יואב / טלי / מיכל / אורי
   - האם יש thumbnail/avatar לכל-קול?
   - כפתור "▶ דוגמה" — איך נראה? איך משתנה במהלך-נגינה?
   - האם המסקוט (Bob) מציג את-עצמו עם הקול?
   - בחירת-קול: איך מסומן הנבחר?

6. **Voice preview animation**:
   - מה קורה ויזואלית כשלוחצים "▶ דוגמה"?
   - הרובוט פותח-פה? יש wave-form?
   - אינדיקטור-נגינה progress?

7. **נגישות (a11y)**:
   - איזה אפשרויות מוצגות?
   - גודל-טקסט dial?
   - dyslexic-font toggle?
   - contrast settings?

8. **Color + visual feel של settings**:
   - דומה לעיצוב-האפליקציה הראשי? או חלק שונה?
   - האם יש קווי-מפריד בין-סקציות?
   - typography של labels

9. **Toast confirmations**:
   - האם יש toast בלחיצה (לדוגמא: "הקול עודכן ל'יואב'")?
   - איך נראה? מאיפה נכנס?

החזר בעברית, מובנה לפי 9 הסעיפים, עם MM:SS לכל-אירוע.

בסוף, חלק:
- "החלק הכי-מעניין של מסך-הקולות לשחזר"
```
