# 🎬 סרטון 02 — Lesson Flow (Quiz Player)

> **תוכן**: שיעור-פעיל ב-StudiesGo עם 4 סוגי-שאלות (MCQ-long / MCQ-short / Matching / Explanation) + feedback-של-תשובה-נכונה/שגויה + תפריט-עליון (XP, streak, progress-dots).
> **משך**: 42 שניות · **גודל**: 17MB

## ⚠️ זה הסרטון הכי-קריטי ל-Phase 5!

הוא חושף את כל אנימציות-ה-feedback שצריך לשחזר.

---

## 📋 פרומפט לג'מיני

```
אני בונה אפליקציית-לימוד בעברית (StudiBuilder), בסגנון Duolingo. אתה מקבל
סרטון screen-recording של מסך-שיעור פעיל באפליקציה דומה (StudiesGo).
הסרטון מראה: MCQ ארוך/קצר, Matching (התאמת-מונחים), Explanation (כרטיס-הסבר),
ומעבר בין תשובה-נכונה לתשובה-שגויה.

אנא תן תיאור-מפורט-בעברית של:

1. **רצף-השאלות בסרטון** (סמן MM:SS):
   - איזה סוג-שאלה מוצג בכל-רגע?
   - איך עוברים בין שאלה לשאלה (slide-up? slide-side? fade?)
   - משך-המעבר (ms)

2. **MCQ-long (4 cards אנכיים)**:
   - מצב-ראשוני: איך נראים ה-4 cards? צבע? border?
   - hover/tap על-card: scale? color-change? border-glow?
   - לאחר-בחירה (לפני-אישור): איך מסומן הנבחר?
   - לאחר-אישור (correct): צבע-ירוק? checkmark animation?
   - לאחר-אישור (wrong): צבע-אדום? shake?

3. **MCQ-short (2×2 grid או fill-blank)**:
   - layout השונה מ-MCQ-long
   - אנימציות-בחירה
   - feedback מתי שנכון/שגוי

4. **Matching (התאמת-זוגות)**:
   - 2 עמודות
   - איך מסמנים בחירה?
   - איך נראית התאמה-מוצלחת? (קו בין הזוגות? color-pair?)
   - מה קורה אם טועים?

5. **Explanation card (טיפ עם נורה)**:
   - איך מופיע? (slide-up? fade?)
   - האם יש הדגשה של הטקסט הקריטי?
   - איך נראה כפתור "המשך"?

6. **Header bar (XP / streak / progress-dots)**:
   - XP-counter: איך מתעדכן אחרי תשובה-נכונה?
     (מספר זז למעלה? +10 floats up? scale-pulse?)
   - Streak fire (🔥): האם מהבהבת? איך?
   - Progress-dots: איך מתמלאים?
     (אחד-בכל-תשובה? איזה צבע? כתום? יש easing?)

7. **תשובה-נכונה — האנימציה המלאה**:
   - תיאור פריים-אחר-פריים (100ms) של 1.5 שניות מהאישור עד למעבר הבא
   - מה צובע ירוק? מה משתנה גודל?
   - האם יש sound effect?
   - האם יש haptic?

8. **תשובה-שגויה — האנימציה המלאה**:
   - shake? באיזו תדירות? כמה זמן?
   - הופעת רובוט עם פוזה-עצובה?
   - הופעת "התשובה הנכונה הייתה..."?
   - כפתור "הסבר לעומק" — מאיפה הוא נכנס?

9. **דמות-המסקוט (Bob - רובוט כחול)**:
   - באיזה מסכים הוא מופיע?
   - idle: האם מצמץ? מתנדנד?
   - תגובה לתשובה-נכונה (פוזה-שמחה?)
   - תגובה לתשובה-שגויה (פוזה-עצובה?)

החזר בעברית מובנה לפי 9 הסעיפים-לעיל, עם MM:SS לכל-אירוע.

בסוף, חלק TL;DR:
- "3 האנימציות הכי-חשובות לשחזר בקוד" (לפי-עדיפות)
- "Easing אופייני לכל-קטגוריה" (snap? bounce? smooth?)
```
