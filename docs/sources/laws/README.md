## פרוטוקול שימוש (Phase 4 — Course Pipeline)

```
[PDF] ← source-of-truth
   ↓ parser (pdf-parse / Claude Vision fallback)
[chunks] עם metadata: { scope_id, page, section_number, paragraph_id }
   ↓ Voyage AI embeddings
[pgvector]
   ↓ user question
[RAG top-3 chunks] ← ציטוט-מילה-במילה מ-PDF
   ↓ Claude Sonnet
[תשובה עם ציטוט + קישור לעמוד-PDF]
```

**אסור**: תשובה ללא citation. אסור: ציטוט מ-`content_scope_extensions.md`. **חייב**: citation למסמך-מקור.

## כשמוסיפים PDF חדש

1. שמור את ה-PDF ב-`docs/sources/laws/<scope-id>-<short-name>.pdf`
2. עדכן את האינדקס למעלה
3. הוסף ב-`content_scope_extensions.md` הפניה: `📎 מקור-PDF: docs/sources/laws/<file>.pdf`
4. בעת ייבוא לפיפליין (Sprint 1+), ה-importer יקרא את ה-PDF, יחתוך לקטעים-עם-מטא-דאטה
5. כל שאלה שתופק מ-PDF זה — תשמור `source_chunk_id` שמצביע על קטע-המקור
