# 📥 Inbox — העלאות-זמניות לפני סיווג

תיקייה זו = **buffer**. אתה מעלה PDFs לכאן (או אלי-בצ'אט ואני שומר לכאן), אני קורא ומסווג.

## תהליך-עבודה

1. **אתה**: drop PDFs ל-`_inbox/` (דרך GitHub web, drag-drop), או שלח אלי בצ'אט
2. **אני**:
   - קורא כל PDF (כותרת, מספר-תקנה, שנה)
   - מזהה scope-ID לפי `docs/content-scope.md`
   - **שואל לאישור** אם יש ספק
   - מעביר לתיקייה-הקטגורית עם שם-תקני: `<scope-id>-<slug>.pdf`
   - מעדכן את `docs/sources/laws/README.md`
3. **commit + push** בסוף

## כללי-שם-תקן

```
<scope-id>-<slug-באנגלית-מקוצר>.pdf
```

דוגמאות:

- `1.5-teunot-machalot-1945.pdf`
- `3.5.1-benzene-1983.pdf`
- `5.1-iso-45001.pdf`

## אם אני לא-מזהה את ה-PDF

אסמן בקובץ-המקור עם תיאור: `_inbox/unknown-<original-name>.pdf` ואשאל אותך:

- מה ה-scope-ID? (אם יש)
- מאיזה נושא?
- האם זה מקור-משלים או חלופי לקיים?

## ניקוי

תיקייה זו צריכה להיות **ריקה** אחרי כל-סשן-סיווג. אם נשאר משהו = ממתין-לאישור.
