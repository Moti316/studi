### [2026-06-08] מנוע-תוכן NotebookLM — מטריצת-תוצרים מלאה (מעבר לתרחישים)

> רעיון-מוטי: "יש כל-כך הרבה דברים שאפשר לשלב!". הגשר (`tools/nblm-bridge/`) פותח את **כל** תוצרי-NotebookLM. **עיקרון-מפתח (היררכיית-אמון):** התשובה-החוקית-הסמכותית = תמיד טקסט-מצוטט-מאומת (שערי-G1–G5 · ציטוט+סעיף). תוצרי-מדיה (פודקאסט/וידאו/שקופיות) = **עזרי-הנעה/לימוד נגזרים** — מעוגנים-במחברת אך לא ניתנים-לאימות-verbatim → מסומנים "סיוע-לימוד מבוסס-AI", לעולם לא מחליפים את האסמכתא-המצוטטת.

| תוצר-NotebookLM (CLI)                                               | בית ב-StudiBuilder                                    | מסילה / ערך                                      | בר-אימות?   |
| ------------------------------------------------------------------- | ----------------------------------------------------- | ------------------------------------------------ | ----------- |
| `generate quiz`                                                     | בנק-MCQ                                               | זורם דרך `buildQuestionRow`+`quoteAppearsInBody` | ✅ G1–G5    |
| `generate report` (study-guide)                                     | הסברים/סיכומי-נושא                                    | `questions.explanation` / חומרי-חזרה             | ✅ ציטוט    |
| `matching` (custom)                                                 | `type=matching` + `MatchingPairs`                     | מושג↔הגדרה · בקרה↔תקנה+סעיף                      | ✅ G1–G5    |
| **`generate flashcards`**                                           | **SM-2 SRS הקיים!** (`src/lib/srs/`)                  | כרטיסיות→חזרה-מרווחת · התאמה-מושלמת              | ✅ טקסט     |
| **`generate audio`** (Audio Overview / פודקאסט)                     | "האזן-לנושא" (commute/חזרה) · מחליף/משלים TTS-Phase-7 | הנעה · נגישות                                    | ⚠️ אות-הנעה |
| **`generate slide-deck`**                                           | מצב-מצגת בשיעור · אתר-קורס/שיווק (Phase-10)           | ויזואל-נושא                                      | ⚠️ נגזר     |
| `generate mind-map`                                                 | עזר-יוצר + תצוגת-לומד                                 | מבנה-דומיין                                      | ⚠️ נגזר     |
| `generate video` / `cinematic-video` / `infographic` / `data-table` | אתר-קורס · נכסי-שיווק · טבלאות-השוואה                 | engagement                                       | ⚠️ נגזר     |

**roadmap מוצע (אחרי הליבה):** ליבה-מאומתת קודם (תרחישים→הסברים→MCQ→matching · G1–G5) → ואז שכבת-מדיה: **flashcards→SM-2** (הכי-זול-וערכי · משתמש בתשתית קיימת) → **פודקאסט פר-נושא** → slide-deck/infographic לאתר-הקורס. כל מדיה = פיילוט-נושא-אחד לאימות-איכות-עברית לפני scale. · ערך: עומק-תוכן + הנעה + נכסי-שיווק · מאמץ: L (מדורג) · 💡 (post-core).
