## 2. יצירת קורס (Core Feature)

### 2.1 שלב 1: בחירת מקור (`/create-c?` Step 1/5)

- **Toggle**: "העלאת קבצים" / "כתיבת טקסט"
- **Drag-and-drop area** + לחצן בחירה
- **סוגי קבצים נתמכים**: PDF, Word, PowerPoint, Excel, תמונות
- **מגבלה**: 50MB לקובץ
- **Picker מובייל**: דיאלוג "בחירת פעולה" עם "תמונות וסרטונים" + "מצלמה"

### 2.2 שלב 2: זיהוי-נושא

- **AI scanning** עם spinner ומסר "בודקים את המקור..."
- **תוצאה**: כרטיס עם תיאור הנושא + **ביטחון % מספרי** ("ביטחון 100%")
- **דוגמת שאלה**: מציג שאלה אחת לדוגמה מהחומר ("שאלה 1: על מי חלה האחריות...")
- **CTA**: "כן, זה הנושא — צור קורס" / "לא — אעלה קובץ אחר"

### 2.3 שלב 3: בחירת תוכן

- **רשימה גרידית** של כל הפריטים שנמצאו ("0 / 570 נבחרו")
- **בחר הכל** / **בטל הכל**
- **כל פריט = כרטיס** עם: מספר, סוג ("מבדק סיכום"), טקסט תקציר
- אפשר לסמן ולסנן

### 2.4 שלב 4: אישור ועלות

- **מסך "הכל מוכן! בוא נוודא שהכל בסדר לפני שמתחילים"**
- **מטריקות**: שם הקורס (יכול לערוך), # קבצים (1), # עמודים (15)
- **Toggle**: "צרף את עמודי המקור לשיעורים" (תראה הסבר ליד כל עמוד מהמקור המדויק)
- **עלות**: ראיתי "8 קרדיטים תוספת" כשמופעל toggle = סה"כ 23 במקום 15
- **יתרה נוכחית** מוצגת
- **CTA**: "צור את הקורס! 🚀"

### 2.5 שלב 5: עיבוד אסינכרוני

- **Pipeline 5-שלבי** עם status: "מעבד.." / "ממתין..":
  1. **Parsing** — חילוץ תוכן מהקבצים
  2. **Chunking** — חלוקה לקטעי לימוד
  3. **RAG** — אינדוקס סמנטי של החומר
  4. **יצירת יחידות ושיעורים**
  5. **יצירת שאלות ותרגולים**
- **Progress bar** ו-% ("ההתקדמות שלי: 5%")
- **ETA**: "כ-12 דקות"
- **חבק-רקע**: "העיבוד ממשיך ברקע — אפשר לסגור את הדפדפן"
- **אופציות**: "צפה בקורס" (תוך כדי), "מעלה קבצים..." (לחיצה לסגירה)

## 3. ניהול שיעור (`/lesson/...`)

### 3.1 Header של שיעור

- **תיוג "שיעור"** מעליה
- **שם השיעור**
- **פס-התקדמות** - חרוזים כתומים (~25 נקודות), בהירים = לא-עברו, כהים = עברו
- **חץ "המשך לשיעור הבא"** מימין
- **XP counter** עם איקון ברק - מצטבר משאלה לשאלה (30 → 110)
- **Streak counter** עם איקון להבה - "3 🔥", "4 🔥"
- **דיסקליימר AI**: "המידע נוצר על-ידי AI ועלול להכיל שגיאות"

### 3.2 סוגי שאלות

**(א) MCQ ארוך** - 4 cards אנכיים בעמודה אחת:
- שאלה ברורה
- כל card מכיל תשובה ארוכה (משפט)
- לחיצה → בחירה (visual highlight)

**(ב) MCQ קצר ("fill in the blank")** - 2x2 grid:
- "המשפט עם _____" - מקום ריק לסימן
- 4 cards עם מילים בודדות

**(ג) Matching pairs** - 2 columns:
- כותרות: "מונחים" (ימין) / "הגדרות" (שמאל)
- 3 תאים בכל עמודה
- בחירה: tap בעמודה אחת ואז בעמודה שניה → צבע-בחירה (כתום)
- "בדוק תשובה" button בתחתית

**(ד) Explanation card** (לא-שאלה):
- Header עם איקון נורה "הסבר"
- כרטיס-טקסט עם הסבר
- כרטיס-פנימי בולט עם "טיפ" (איקון נורה צהובה)
- "הבנתי, המשך" כפתור כחול

### 3.3 פידבק לאחר תשובה

**תשובה נכונה:**
- צ'ק ירוק (לא ראיתי במפורש אבל סביר)
- XP מתעדכן
- כפתור "המשך" כחול

**תשובה לא נכונה:**
- מסך-פידבק עם הרובוט + "תשובה לא נכונה"
- **טבלת-התאמות** עם כל ההתאמות שלך, ה-X-ים האדומים והצ'ק-ים הירוקים
- "התשובה הנכונה: ..."
- **תיבת הסבר** עם תיאור התשובות הנכונות
- **כפתור "הסבר לעומק"** (קסם-AI) משמאל
- **כפתור "המשך"** מימין

### 3.4 איקוני-כותרת בשיעור

- 🚩 דגל אדום - דווח-בעיה / "טעות בשאלה"
- 🔊 רמקול - הקראה (toggle?)
- 💡 נורה (בלוק "הסבר") - הצג/הסתר הסבר

## 5. הגדרות (`/settings`)

### 5.1 חשבון

- שם תצוגה
- (לא ראיתי אבל סביר): שינוי מייל, סיסמה

### 5.2 למידה

- **יעד יומי**: 4 כפתורי-בחירה: 5 / 10 / **15** / 20 דקות. כתום = נבחר. תווית "מאוזן" לבחירה האמצעית

### 5.3 מראה (תצוגה)

- **ערכת נושא**: 3 כפתורים גדולים:
  - מערכת 🖥
  - **כהה** 🌙 (נבחר במסגרת סגולה)
  - בהיר ☀

### 5.4 קול

- **toggle "השמע את ההסברים בקול"** - כשמופעל ההסבר של כל שיעור מוקרא אוטומטית
- **"קול ההקראה של בוב"** - 4 כפתורי-בחירה:
  - יואב (גברי, חם) - נבחר ✓
  - טלי (נשי, בהיר)
  - מיכל (נשי, חם)
  - אורי (גברי, צעיר)
  - כל אחד עם כפתור **▶ דוגמה**
- **Toast confirmation** בתחתית: "הקול עודכן ל'יואב'" עם ✓

### 5.5 נגישות

- **toggle "הצג את כפתור נגישות בכל עמוד"** - כפתור-צף שפותח תפריט נגישות (גודל גוף, ניגודיות, הפסקת אנימציות וכו')
- **toggle "הצג את הכפתור גם במובייל"** - דיפולט סגור במובייל

### 5.6 התראות

- **toggle "מיילים שיווקיים"** - עדכונים על פיצ'רים חדשים, טיפים, ותוכן מיוחד

### 5.7 עזרה ופעולות-חשבון

- **"צור קשר עם הצוות"** - תמיכה
- **התנתק** (כפתור)
- **מחק חשבון** (כפתור)

## 8. בוב - המסקוט/המנחה

הרובוט הכחול שמופיע בכל מקום הוא דמות מרכזית:
- **בדשבורד** - יושב ליד "+ קורס חדש"
- **בכניסה ליצירה** - "סקרני" (פוזה ראשונה)
- **בעיבוד** - "מתרכז" (מסתכל קדימה)
- **בתום עיבוד** - "שמח" (פנים מחייכות)
- **בפידבק שלילי** - לא-מאוכזב, פותח דיאלוג עם הסבר
- **שמו**: "בוב" — בהגדרות נכתב "קול ההקראה של בוב". נותן זהות אנושית-חמה למערכת AI

## 9. AI / LLM features

- **זיהוי נושא** (Q→A) — מהקובץ מזהה תחום ידע
- **חילוץ-חומר** (PDF parsing → text)
- **Chunking סמנטי** — חלוקה לקטעי-לימוד
- **RAG indexing** — vector DB
- **יצירת יחידות-לימוד** מתוך chunks
- **יצירת שאלות** מסוגים שונים (MCQ / matching / fill-blank / explanation)
- **TTS** עברית (4 קולות)
- **"הסבר לעומק"** — סביר LLM-call על דרישה, עם RAG-context מהחומר המקורי
