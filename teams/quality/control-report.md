# דו"ח-בקרה — צוות `quality` (איכות)

> דו"ח-בקרה **מתגלגל** מראש-הצוות (מירב / `quality-lead`) למתווך (אמיר / `mediator`).
> מאחד את דיווחי חמשת חברי-הצוות (`appsec`, `content-verifier`, `e2e-qa`,
> `privacy-officer`, `test-engineer`) לכדי סטטוס-איכות אחד + סטטוס-דגלים.
> מתעדכן בכל סוף-מחזור. מבנה-הדיווח: [`../ORG.md`](../ORG.md) §קצב-דיווח.

> פורמט-רשומה: `## [תאריך שעה] מחזור/נושא` ואז —
> סטטוס-צוות · דיווחי-חברים (slug → Outcome + Verdict) · דגלים פתוחים ·
> הסלמות-למתווך · החלטות-שער-איכות (PASS / CONCERNS / FAIL).
